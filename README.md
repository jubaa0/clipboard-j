تطبيق مدير حافظة (Clipboard Manager) لـ Linux مبني على GTK4، يدعم حفظ وإدارة النصوص والصور والملفات (حتى 50 عنصر افتراضيًا، قابل للتعديل)، مع ميزات التثبيت، الحذف، المسح، اختصار عالمي (افتراضيًا Ctrl+Alt+V، قابل للتغيير عبر tray)، حفظ في JSON، وأيقونة tray للتحكم. يراقب الحافظة باستمرار ويظهر نافذة popup عند الضغط على الاختصار.
